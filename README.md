# PRODIGY_DS_02
Task 2: Data Cleaning and Exploratory Data Analysis (EDA)

Objective

Perform data cleaning and exploratory data analysis (EDA) on the Titanic dataset to identify patterns, relationships, and trends in the data.

Dataset

Titanic Dataset

Tools Used
	•	Python
	•	Pandas
	•	Matplotlib
	•	Jupyter Notebook

Tasks Performed
	•	Loaded and explored the dataset.
	•	Checked for missing values and cleaned the data.
	•	Analyzed different features using descriptive statistics.
	•	Visualized relationships between variables using charts and graphs.
	•	Identified trends and patterns affecting passenger survival.

Key Insights
	•	Missing values were identified and handled appropriately.
	•	Survival rates varied based on factors such as gender, age, and passenger class.
	•	Data visualization helped in understanding important relationships between variables.

Outcome

Successfully cleaned the dataset and performed exploratory data analysis to gain meaningful insights into the Titanic dataset.

Learning
	•	Data Cleaning Techniques
	•	Exploratory Data Analysis (EDA)
	•	Handling Missing Values
	•	Data Visualization using Matplotlib
	•	Working with Real-World Datasets
